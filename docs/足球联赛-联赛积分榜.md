2、联赛积分榜 [顶部]
接口地址： https://apis.juhe.cn/fapig/football/rank
返回格式： json
请求方式： http get/post
请求示例： https://apis.juhe.cn/fapig/football/rank
接口备注： 根据联赛类型,查询积分榜排名。数据来源于网络整理，可能会有一定误差和延时。
请求Header：
名称 值
Content-Type application/x-www-form-urlencoded
请求参数说明：
名称 类型 必填 说明
key string 是 在个人中心->我的数据,接口名称上方查看
type string 是
足球联赛类型, 可选:
jiangsu(江苏城市足球联赛、苏超)
xijia(西甲)
dejia(德甲)
yingchao(英超)
yijia(意甲)
fajia(法甲)
zhongchao(中超)
返回参数说明：
名称 类型 说明
error_code int 返回码
reason string 返回说明
result obj 返回结果集
title string 足球联赛名称
duration string 联赛赛季
ranking array 联赛积分排名情况
rank_id string 排名
team string 球队名称
wins string 胜场
losses stirng 败场
draw string 平局
goals string 进球数, 可能为空
losing_goals string 失球数,可能为空
goal_difference string 净胜球
scores string 积分

2、错误码参照

209001 不支持的联赛类型
209002 请求数据源异常
209003 查询不到相关数据

10007 未知的请求源 107

10009 被禁止的KEY 109
